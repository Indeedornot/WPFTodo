## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/indeedornot/WPFWeather.git
   ```
3. Build the project
   ```sh
   dotnet build
   ```
4. Run the project
   ```js
   dotnet run
   ```
